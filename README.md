# House Price Predictor

A machine learning web application that predicts house prices based on:

- Overall Quality
- Living Area
- Garage Capacity
- Basement Area

## Technologies Used

- Python
- Scikit-Learn
- Pandas
- NumPy
- Matplotlib
- Gradio

## Model Performance

- R² Score: 0.79
- MAE: 25,446

## Features

- Interactive Gradio UI
- Price Prediction
- House Category Classification
- Prediction Visualization
